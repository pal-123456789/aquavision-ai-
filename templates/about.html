About AquaVision AI

AquaVision AI is a portfolio project designed to demonstrate a full-stack web application capable of analyzing environmental data.

The Mission

The goal of this project is to simulate the detection and prediction of Harmful Algal Blooms (HABs) using publicly available concepts from NASA satellite data. It serves as a proof-of-concept for how modern cloud technologies can be leveraged for environmental monitoring.

Technology Stack

This application was built with a range of modern technologies to showcase a complete development and deployment pipeline:

  • Backend: Python with the Flask framework.
  • Data Processing: NumPy, OpenCV, Rasterio, Pandas.
  • Frontend: HTML, JavaScript, and the Bootstrap 5 framework for a responsive UI.
  • Mapping: Leaflet.js with Esri World Imagery tiles.
  • Deployment: The entire application is containerized using Docker and deployed on Render's cloud platform, featuring a full CI/CD pipeline linked to GitHub.
Disclaimer

This is a personal, non-commercial project for educational and demonstration purposes. The data analysis presented is simulated and should not be used for real-world navigation, safety, or decision-making.
